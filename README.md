# Social Distancing
Social distancing and mask detection

### How to run
1. Clone the repo
2. Install docker for model server
3. Install requirements `pip install -r requirements.txt`
4. Run main.py </br>
    - `python3 main.py` </br>
    - `python3 main.py -i test.mp4 -o out.avi`
